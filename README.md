# line-getter
## About
- line-getter is a VSCode extension.
- Right-click the line number in the editor and select `Copy Path with Line Number` to copy the currently selected filename and line number to the clipboard.
- In debuggers for languages like Python and C++, setting breakpoints requires specifying the filename and line number. This extension makes the process smooth and effortless.

# line-getter
## 概要
- line-getter は VSCode の拡張機能です。
- エディタの行番号を右クリックし、`Copy Path with Line Number` をクリックすると現在選択しているファイル名と行番号をクリップボードにコピーできます。
## 使用例
- Python や C++ などのデバッガにおいて、ブレークポイントを設定するにはファイル名と行番号が必要です。そこでこの拡張機能を使用するとスムーズに設定できます。
